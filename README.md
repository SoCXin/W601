### [简介](https://github.com/tfzoo/cluster/wiki) 

聚类是一种无监督学习任务，该算法基于数据的内部结构寻找观察样本的自然族群（即集群）。

因为聚类是一种无监督学习（数据无标注），通常使用数据可视化评价结果。
